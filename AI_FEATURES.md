# 🤖 AI ФУНКЦІЇ UKROVERFLOW 2.0

## Powered by Google Gemini 2.5 Flash & Gemini 2.5 Pro

UkrOverflow тепер має 5 потужних AI-функцій, інтегрованих через Google Gemini API з найновішими моделями.

---

## 📋 СПИСОК AI ФУНКЦІЙ

### 1️⃣ AI-ПОМІЧНИК ДЛЯ ВІДПОВІДЕЙ (Gemini Pro)
**Де:** На сторінці питання, під формою відповіді
**Модель:** Gemini 2.5 Pro
**Що робить:**
- Аналізує питання (заголовок, тіло, теги)
- Генерує детальну структуровану відповідь
- Включає пояснення, покрокові рішення, приклади коду
- Підказка з'являється в спеціальному блоці з можливістю використати або відхилити

**Як користуватися:**
1. Відкрийте будь-яке питання
2. Прокрутіть до форми "ВАША ВІДПОВІДЬ"
3. Натисніть кнопку **✨ AI ПІДКАЗКА**
4. Зачекайте 3-5 секунд
5. Натисніть **✓ ВИКОРИСТАТИ** щоб вставити текст у редактор
6. Відредагуйте підказку перед публікацією

**Приклад використання:**
```
Питання: "Як налаштувати React Router в Vite?"
AI згенерує: Детальну відповідь з кроками встановлення,
конфігурації, прикладами коду та порадами щодо best practices.
```

---

### 2️⃣ AUTO-TAGGING ПИТАНЬ (Gemini Flash)
**Де:** На сторінці створення питання, під полем тегів
**Модель:** Gemini 2.5 Flash
**Що робить:**
- Аналізує заголовок та опис питання
- Пропонує 3-5 найбільш релевантних тегів
- Автоматично підбирає технології, мови програмування, фреймворки

**Як користуватися:**
1. Створіть нове питання (введіть заголовок та опис)
2. Натисніть кнопку **🏷️ AI ТЕГИ** під полем тегів
3. AI запропонує релевантні теги
4. Натисніть на окремий тег щоб додати його
5. Або натисніть **✓ ОБРАТИ ВСІ** щоб додати всі

**Приклад:**
```
Заголовок: "Помилка з async/await в Node.js Express"
Опис: "Під час виконання асинхронного запиту до БД..."

AI запропонує: javascript, node.js, express, async-await, mysql
```

### 3️⃣ МОДЕРАЦІЯ КОНТЕНТУ (Gemini Flash)
**Де:** Backend, працює автоматично
**Модель:** Gemini 2.5 Flash
**Що робить:**
- Автоматично перевіряє нові питання та відповіді
- Виявляє SPAM (реклама, непов'язаний контент)
- Виявляє TOXIC (образи, hate speech)
- Виявляє LOW_QUALITY (надто короткі, незрозумілі)
- Позначає підозрілий контент для модераторів

**Статуси:**
- ✅ **OK** - нормальний контент, публікується
- ⚠️ **LOW_QUALITY** - попередження, але публікується
- 🚫 **SPAM** - блокується автоматично
- 🚫 **TOXIC** - блокується автоматично

**Примітка:** Модерація працює прозоро для користувача.

---

### 4️⃣ SMART SUMMARIZATION (Gemini Flash)
**Де:** На сторінці питання, над текстом питання
**Модель:** Gemini 2.5 Flash
**Що робить:**
- Автоматично генерує TL;DR для питань довше 500 символів
- Створює коротке резюме (2-3 речення)
- Показує суть проблеми без читання всього тексту

**Як працює:**
- Автоматично завантажується при відкритті питання
- З'являється жовтий блок **🤖 TL;DR (AI)** над текстом
- Якщо питання коротке (<500 символів), резюме не створюється

**Приклад:**
```
Довге питання про налаштування WebSocket з'єднання...

TL;DR (AI): Користувач намагається налаштувати WebSocket
сервер на Express, але отримує помилку CORS. Потрібна
допомога з конфігурацією middleware та headers.
```

---

### 5️⃣ ПОШУК СХОЖИХ ПИТАНЬ (Gemini Flash)
**Де:** На сторінці питання, над формою відповіді
**Модель:** Gemini 2.5 Flash
**Що робить:**
- Автоматично шукає семантично схожі питання
- Не лише за ключовими словами, але і за сенсом
- Показує топ-3 найбільш релевантних питань
- Допомагає уникнути дублювання питань

**Як працює:**
- Автоматично завантажується при відкритті питання
- Блок **🤖 СХОЖІ ПИТАННЯ (AI)** з'являється автоматично
- Клік на питання відкриває його
- Показує статистику (перегляди, відповіді, голоси)

**Приклад:**
```
Ви запитуєте: "Як зробити API сервер на Node.js?"

AI знайде схожі:
- "Express backend для початківців"
- "REST API з Node та MySQL"
- "Налаштування Express.js проекту"
```

## 🚀 НАЛАШТУВАННЯ

### 1. Отримати Gemini API ключ

1. Перейдіть на https://aistudio.google.com/apikey
2. Увійдіть через Google акаунт
3. Натисніть **Create API Key**
4. Скопіюйте ключ

### 2. Додати ключ у .env

Відредагуйте файл `.env` у кореневій папці:

```bash
GEMINI_API_KEY=ваш_ключ_тут
```

Або використайте наявний:
```bash
GEMINI_API_KEY=your_gemini_api_key_here
```

### 3. Встановити залежності

```bash
cd packages/backend
npm install
```

### 4. Перезапустити сервер

```bash
npm run dev
```

## 🎯 API ЕНДПОІНТИ

Всі AI функції доступні через REST API:

### POST /api/ai/suggest-answer
Генерує AI-підказку для відповіді
```json
Request: { "questionId": 1 }
Response: {
  "success": true,
  "data": {
    "suggestion": "Детальна відповідь...",
    "model": "gemini-1.5-pro"
  }
}
```

### POST /api/ai/suggest-tags
Пропонує теги для питання
```json
Request: { "title": "...", "body": "..." }
Response: {
  "success": true,
  "data": {
    "tags": ["javascript", "react", "hooks"],
    "model": "gemini-2.0-flash"
  }
}
```

### POST /api/ai/moderate
Модерує контент
```json
Request: { "text": "...", "type": "question" }
Response: {
  "success": true,
  "data": {
    "status": "OK",
    "isAcceptable": true,
    "model": "gemini-2.0-flash"
  }
}
```

### POST /api/ai/summarize
Створює резюме питання
```json
Request: { "questionId": 1 }
Response: {
  "success": true,
  "data": {
    "needsSummary": true,
    "summary": "Коротке резюме...",
    "model": "gemini-2.0-flash"
  }
}
```

### GET /api/ai/similar-questions/:questionId
Знаходить схожі питання
```json
Response: {
  "success": true,
  "data": {
    "similarQuestions": [...],
    "model": "gemini-2.0-flash"
  }
}
```

### GET /api/ai/status
Перевіряє статус AI сервісу
```json
Response: {
  "success": true,
  "data": {
    "available": true,
    "models": {
      "flash": "gemini-2.5-flash",
      "pro": "gemini-2.5-pro"
    },
    "features": [...]
  }
}
```

## 🛠️ TROUBLESHOOTING

### AI функції не працюють

1. **Перевірте API ключ:**
   ```bash
   # У .env файлі
   GEMINI_API_KEY=your_key_here
   ```

2. **Перевірте інтернет-з'єднання:**
   AI потребує з'єднання з Google API

3. **Перевірте консоль браузера:**
   Відкрийте DevTools (F12) → Console

4. **Перевірте backend логи:**
   ```bash
   cd packages/backend
   npm run dev
   # Дивіться вивід в консолі
   ```

### Помилка "API key not valid"

- Переконайтеся що ключ правильний
- Спробуйте створити новий ключ на https://aistudio.google.com/apikey
- Перевірте що немає зайвих пробілів у .env файлі

### Помилка "Rate limit exceeded"

- Зачекайте 1 хвилину
- Безкоштовний ліміт: 15 req/min для Flash, 2 req/min для Pro
- При необхідності оновіть тарифний план

### AI генерує неякісні відповіді

- AI підказки потрібно **редагувати** перед публікацією
- Це нормально - AI дає базу, ви покращуєте
- Чим детальніше питання, тим краща підказка
